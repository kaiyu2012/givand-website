# Givand Logo Files

## Logo Design

Your Givand logo features an **industrial warehouse buildings illustration** that represents your building materials import and export business.

### Design Elements

- **Three warehouse buildings** - Shows different types of industrial structures
- **Left building** - Warehouse with garage door (represents loading/storage)
- **Center building** - Tall office/factory building with windows
- **Right building** - Large warehouse with grid windows and cargo boxes
- **Blue gradient** - Professional color scheme (#2563eb to #1d4ed8)
- **"GIVAND"** text with "IMPORT & EXPORT" tagline

## Available Logo Files

All logos are in the `public/` folder:

### 1. **favicon.svg** (64x64px)
- Browser tab icon
- Simplified warehouse illustration on blue gradient background
- Automatically displays in browser tabs and bookmarks

### 2. **logo-icon.svg** (120x80px)
- Warehouse buildings only (no text)
- Used in website header and footer
- Perfect for small spaces or when text is separate

### 3. **logo.svg** (280x160px)
- Vertical layout
- Warehouse illustration + "GIVAND" + "IMPORT & EXPORT"
- Good for documents, presentations, vertical spaces

### 4. **logo-horizontal.svg** (320x90px)
- Horizontal layout
- Warehouse on left, text on right
- Perfect for headers, banners, email signatures
- Most versatile version

## Usage on Website

Currently used in:

## File Format

All logos are **SVG (Scalable Vector Graphics)**:
- ✅ Scale to any size without losing quality
- ✅ Small file size for fast loading
- ✅ Perfect for web, print, and digital use
- ✅ Editable in vector software (Adobe Illustrator, Figma, etc.)

## Exporting to Other Formats

If you need PNG, JPG, or PDF versions:

**Online conversion (easiest):**
1. Visit: https://cloudconvert.com/svg-to-png
2. Upload the SVG file
3. Select output format (PNG, JPG, PDF)
4. Download the converted file

**Recommended sizes for export:**
- Social media profile: 512x512px
- Business card: 300 DPI at actual size
- Letterhead: 300 DPI, 2-3 inches wide
- Banners/Large format: Keep as SVG (infinitely scalable)
